# ربات تلگرام سنگ کاغذ قیچی (اینلاین)

## راه‌اندازی

### ۱) نصب وابستگی‌ها
```bash
pip install -r requirements.txt
```

### ۲) ساخت ربات در تلگرام
1. به [@BotFather](https://t.me/BotFather) پیام بده و با `/newbot` یک ربات جدید بساز.
2. توکنی که می‌دهد را کپی کن.
3. حتماً حالت اینلاین را برای ربات فعال کن:
   - به BotFather پیام بده: `/setinline`
   - ربات خودت را انتخاب کن
   - یک متن placeholder وارد کن (مثلاً: "برای بازی کلیک کن")

### ۳) قرار دادن توکن
فایل `bot.py` را باز کن و این خط را ویرایش کن:
```python
BOT_TOKEN = "PUT_YOUR_BOT_TOKEN_HERE"
```
توکن واقعی‌ای که از BotFather گرفتی را به جایش بگذار.

### ۴) اجرای ربات
```bash
python3 bot.py
```

ربات با polling اجرا می‌شود و دیتابیس SQLite (`rps_games.db`) به‌صورت خودکار در همان پوشه ساخته می‌شود. اگر ربات ری‌استارت شود، بازی‌های قبلی (وضعیت، امتیاز، شرکت‌کننده‌ها) از بین نمی‌روند.

## نحوه استفاده

1. در هر چتی (خصوصی، گروه، سوپرگروه) تایپ کن: `@نام_ربات_شما`
2. روی نتیجه‌ی «🎮 بازی سنگ کاغذ قیچی» که ظاهر می‌شود کلیک کن.
3. پیام بازی با دکمه‌ی «منم می‌خوام بازی کنم 🎮» در همان چت ارسال می‌شود؛ فرستنده به‌عنوان شرکت‌کننده‌ی اول ثبت می‌شود.
4. هر کاربر دیگری که در همان چت روی «منم می‌خوام بازی کنم» بزند، به‌عنوان شرکت‌کننده‌ی دوم ثبت می‌شود.
5. فقط شروع‌کننده می‌تواند دکمه‌ی «شروع 🚀» را بزند.
6. بعد از شروع، هر دو بازیکن دکمه‌های سنگ / کاغذ / قیچی را می‌بینند. هرکس روی دکمه‌ای که می‌زند، فقط خودش (با popup تلگرام) می‌بیند که چه انتخابی ثبت شده — حریف از انتخابش خبردار نمی‌شود تا هر دو انتخاب کنند.
7. وقتی هر دو نفر انتخاب کردند، نتیجه‌ی آن دست در پیام اصلی اعلام می‌شود.
8. بعد از پایان دست، یکی از بازیکنان دکمه‌ی «دست بعد ▶️» را می‌زند تا دست بعدی شروع شود.
9. بعد از مجموع ۳ دست، برنده‌ی نهایی اعلام می‌شود.
10. به‌محض پایان هر بازی (در هر گروهی که اتفاق بیفتد)، خلاصه‌ی کامل بازی (انتخاب هر دست + برنده‌ی نهایی) به پیوی آیدی عددی مشخص‌شده در متغیر `ADMIN_REPORT_ID` ارسال می‌شود.

## نکات فنی

- **پایداری داده**: همه‌ی وضعیت بازی‌ها در SQLite (`rps_games.db`) ذخیره می‌شود، بنابراین با ری‌استارت ربات از بین نمی‌روند.
- **منصفانه بودن بازی**: انتخاب هر بازیکن فقط برای خودش، از طریق `answer(..., show_alert=True)` تلگرام، نمایش داده می‌شود. هیچ کاربری (از جمله ادمین) انتخاب حریف را پیش از ثبت انتخاب خودش نمی‌بیند؛ این یک طراحی عمدی برای حفظ صداقت بازی است.
- **گزارش ادمین**: گزارش فقط بعد از پایان کامل بازی (بعد از هر ۳ دست) ارسال می‌شود، نه در حین بازی — یعنی این گزارش امکان تقلب در حین بازی نمی‌دهد، صرفاً یک خلاصه‌ی نهایی برای پایش است.
- اگر می‌خواهی ربات روی سرور ۲۴ ساعته اجرا شود (به‌جای فقط روی کامپیوتر شخصی)، نیاز به یک VPS یا سرویس میزبانی (مثل Railway, Render, یا یک سرور لینوکسی شخصی) داری و باید با systemd یا screen/tmux آن را به‌صورت دائمی در حال اجرا نگه داری.
